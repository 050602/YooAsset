# 调试器

调试器是在游戏运行时，帮助我们查看资源包加载信息的工具。可以查看资源对象列表信息（AssetView），资源包列表信息（BundleView）。

未来计划支持在编辑器下通过远程调试真机信息。

**注意**：该工具仅支持Unity2019+
